## 🚀 Quick Commands

### Start Development
```bash
# Terminal 1 - Backend
cd backend
npm install
npm run dev

# Terminal 2 - Frontend
cd frontend
npm install
npm run dev

# Then open: http://localhost:3000
```

### Build for Production
```bash
cd frontend
npm run build
# Creates 'dist' folder
```
