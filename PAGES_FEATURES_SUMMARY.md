## 🎯 User Journeys

### New Customer Journey:
1. Lands on Home page
2. Views hero video and testimonials
3. Clicks "Explore Services"
4. Filters services by duration
5. Clicks "Book Now" on preferred service
6. Completes 3-step booking process
7. Receives confirmation email

### Information Gathering:
1. Home page → Services (explore options)
2. Services → About (learn about team)
3. About → Contact (ask questions)
4. Contact → Booking (finalize)

### Quick Booking:
1. Home page → "Book Now" button
2. Booking page (pre-filled if service selected)
3. 3-step process
4. Confirmation
