## 🔧 Technical Details

### Frontend Stack:
- **Framework:** React 18 with Vite
- **Routing:** React Router v6
- **Styling:** Tailwind CSS with custom config
- **Animations:** AOS (Animate On Scroll)
- **Icons:** React Icons (Font Awesome, etc.)
- **Form Handling:** Controlled components
- **SEO:** React Helmet for meta tags
- **Dark Mode:** React Context API

### Backend Ready:
- API endpoints prepared for:
  - Contact form submissions
  - Future booking system
  - Email notifications
- Express.js backend structure in place

### Performance Optimizations:
- Lazy loading on images (`loading="lazy"`)
- Smooth scroll behavior
- Code splitting with React Router
- CSS purging with Tailwind
- Minification on build
- Optimized animations (using transforms/opacity)
